# Project Brief: BLKOUT Energy Drink Landing Page

## Overview
BLKOUT Energy Drink is a product launch landing page for an energy drink brand. The design language is aggressive, neon-themed, dark, and dynamic. It features a single-page layout with 6 sections, glitch effects, neon pulse animations, and fast transitions.

## Core Requirements
- Dark theme with neon accent colors (green, pink, blue)
- 6-section single page layout
- Glitch text reveal animations
- Neon particle canvas background
- Interactive neon card components
- Smooth scroll experience
- Mobile responsive design

## Target Audience
Young, energetic demographic interested in energy drinks, gaming, nightlife, and extreme sports.

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- GSAP + @gsap/react
- Framer Motion
- Lenis (smooth scroll)
- clsx

## Success Criteria
- Zero build errors
- Smooth 60fps animations
- Responsive across all device sizes
- Fast initial load time
